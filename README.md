我在这里公开我真正会反复使用的 AI 协作方法、Skills 和工作流。

English: I publish practical AI collaboration workflows, reusable skills, and learning loops. If you are new here, start with `ai-collab-playbook` and `completion-learn`.

我当前最关心的不是“再堆更多工具”，而是把这三件事逐步做强：

1. 先把人自己变强
2. 再把人和工具的协作做顺
3. 最后再让工具栈持续进化

如果你是第一次来到这里，最推荐从下面三条路径进入。
